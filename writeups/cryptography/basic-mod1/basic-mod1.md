# picoCTF 2022 - basic-mod1

## 基本資訊
- **題目名稱**: basic-mod1
- **平台 / 年份**: picoCTF 2022
- **分類**: Cryptography
- **難度**: Medium

## 題目描述
題目提供一份數字訊息檔案，並給出解密規則：

- 將每個數字對 37 取模
- 0–25 對應大寫英文字母 `A–Z`
- 26–35 對應十進位數字 `0–9`
- 36 對應底線 `_`

最後再將解出的內容包成 picoCTF flag 格式：

`picoCTF{decrypted_message}`

## 解題觀察
這題的核心是模運算（modulo）與字元映射。

注意題目中的 `mod 37` 不是「除以 37 取商」，而是：

`n % 37`

也就是**除以 37 後取餘數**，再把餘數依照題目給的規則映射成字元。

## 核心觀念
本題主要考查兩個觀念：

1. **模運算**
   - `n % 37` 表示除以 37 後的餘數
   - 不是取商

2. **字元映射**
   - 餘數落在不同區間時，要轉成不同型別的字元
   - 0–25 是字母，26–35 是數字，36 是底線

## Python 範例
以下是可重現此題解法的簡單 Python 程式：

```python
nums = [350, 63, 353, 198, 114, 369, 346, 184, 202, 322, 94, 235, 114, 110, 185, 188, 225, 212, 366, 374, 261, 213]

result = []

for n in nums:
    r = n % 37
    if 0 <= r <= 25:
        result.append(chr(ord('A') + r))
    elif 26 <= r <= 35:
        result.append(str(r - 26))
    else:
        result.append('_')

decoded = ''.join(result)
print(decoded)
print(f"picoCTF{{{decoded}}}")
```

## 心得
這題難度不高，但很適合熟悉模運算與字元映射的基本流程。只要正確理解 `mod 37` 是「取餘數」，後面的解法就相當直接。這類題目常見於初階密碼學練習，重點在於準確理解題意並將規則完整落實。

## 重點整理
1. `mod 37` = 對 37 取餘數  
2. 先取餘數，再查表轉字元  
3. 0–25 對應 `A-Z`  
4. 26–35 對應 `0-9`  
5. 36 對應 `_`
